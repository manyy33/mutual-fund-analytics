# mutual# Mutual Fund Analytics Project


## Objective

To analyze mutual fund performance, NAV trends, and investor transactions using Python, Pandas, SQLite and SQL.


## Tools Used

- Python 3
- Jupyter Notebook
- Pandas
- SQLite
- SQLAlchemy


## Project Workflow

1. Data Collection
2. Data Cleaning
3. Database Design
4. Data Loading
5. SQL Analysis


## Database Schema

The project follows a Star Schema:

### Dimension Tables
- dim_fund
- dim_date


### Fact Tables
- fact_nav
- fact_transactions
- fact_performance


## Key Analysis

- Top performing mutual funds
- Category-wise returns
- Investor transaction analysis
- NAV trend analysis


## Output

Cleaned datasets and SQL query results are stored in the output folder.
